逻辑运算符

逻辑运算符包括 && （与） || （或） ！（非） 逻辑操作元必须是boolean型

短路原理

“&&”运算符检查第一个表达式是否返回“false”，如果是“false”则结果必为“false”，不再检查其他内容。
“a/0”是个明显的错误！但短路运算&&先判断a < b,返回“false”，遂造成短路，也就不进行“a/0”操作了，程序会打出”That’s in my control.”。这个时候，交换一下“&&”左右两边的表达式，程序立即抛出异常“java.lang.ArithmeticException: / by zero”。

  3. 局部变量
    在同一个方法中，不允许有同名局部变量
    在不同的方法中 可以
    在同一个类中，成员变量和局部变量同名时，局部变量具有更高优先级

  4. 成员变量
    全局变量

    成员变量与局部变量的区别

  5. 作用域不同
    局部变量的作用域仅限于定义它的方法
    成员变量的作用域在整个类内部都是可用的

  6. 初始值不同
    java 会给成员变量一个初始值
    java 不会给局部变量赋予初始值
